# Bible Deep Learning

Interactive web-based guide for deep learning concepts with illustrated chapters.

## Overview

A comprehensive, illustrated HTML book that explains deep learning concepts chapter by chapter. Each chapter is a standalone HTML page with visual explanations, CSS styling, and JavaScript interactivity.

## Contents

- `index.html` -- Main entry point and table of contents
- `pages/` -- Individual chapter HTML files
- `css/` / `js/` -- Styling and interactive scripts
- `book_page_jpg/` -- Scanned book page images

## Getting Started

Open `index.html` in a browser to start reading. No build step required.
