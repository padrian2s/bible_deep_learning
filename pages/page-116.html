import numpy as np from sklearn.linear_model import LinearRegression # Predictia pretului unei case bazat pe features # Features: [suprafata_mp, nr_camere, varsta_ani, distanta_centru_km] X_train = np.array([ [50, 2, 10, 5], [80, 3, 5, 3], [120, 4, 2, 8], [60, 2, 15, 2], ]) y_train = np.array([150000, 250000, 350000, 180000]) # Preturi in EUR model = LinearRegression() model.fit(X_train, y_train) # Predictie pentru o casa noua casa_noua = [[100, 3, 8, 4]] pret_prezis = model.predict(casa_noua)[0] print(f"Casa: 100mp, 3 camere, 8 ani, 4km de centru") print(f"Pret prezis: {pret_prezis:,.0f} EUR") print(f"\nCoeficienti (importanta features):") for feat, coef in zip(['Suprafata', 'Camere', 'Varsta', 'Distanta'], model.coef_): print(f" {feat}: {coef:+.0f} EUR per unitate")

Transcriptie

In acest task, sistemul ML observa o reprezentare relativ nestructurata a datelor si trebuie sa o transcrie intr-o forma discreta, textuala. Exemple: OCR (optical character recognition) - unde input-ul este o fotografie cu text si output-ul este o secventa de caractere; speech recognition - unde input-ul este o forma de unda audio si output-ul este secventa de cuvinte rostite. Google Street View foloseste deep learning pentru a procesa numere de adrese din fotografii.

Traducere Automata (Machine Translation)

In machine translation, input-ul este deja o secventa de simboluri intr-o limba, iar programul trebuie sa o converteasca intr-o secventa de simboluri in alta limba. Aplicat frecvent pentru traducerea limbajelor naturale, cum ar fi traducerea din engleza in franceza. Deep learning a inceput recent sa aiba un impact important asupra acestui task.

üìö
Arhitecturi Seq2Seq ‚ñ∂
Sutskever et al. (2014) - Sequence to Sequence Learning with Neural Networks
Bahdanau et al. (2015) - Attention mechanism pentru traducere - revolutionar!

Output Structurat (Structured Output)

Task-urile cu output structurat implica output-uri care sunt vectori sau structuri de date cu relatii importante intre elemente. Aceasta categorie larga include transcriptia si traducerea. Un exemplu este parsing - transformarea unei propozitii intr-un arbore gramatical. Alt exemplu este segmentarea imaginilor - unde programul asigneaza fiecare pixel unei categorii specifice.
