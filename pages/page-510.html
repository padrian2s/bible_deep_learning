Deep SFA: Features Biologic Plausibile

Deep SFA (SFA aplicat pe video-uri de scene naturale) invata features care seamana cu cele din cortexul V1 (Berkes & Wiskott, 2005) si din hipocampul sobolanilor (Franzius et al., 2007) - place cells pentru navigare!

Avantajul Teoretic al SFA

Un avantaj major: pentru SFA, putem prezice teoretic ce features va invata modelul! Daca stim dinamica mediului (cum se schimba factorii in timp), putem deriva analitic features-urile optime.

Spre deosebire de deep learning unde features depind de valori specifice ale pixelilor si sunt greu de prezis, SFA depinde doar de dinamica mediului.

Dezavantaj: slowness prior poate fi prea puternic - ignora obiectele cu viteza mare chiar daca sunt importante.

Aplicatii Practice

Deep SFA a fost folosit pentru object recognition si pose estimation (Franzius et al., 2008), desi nu a devenit state-of-the-art. Principiul slowness ramane relevant in multe contexte.
