Proprietatile Inmultirii Matriciale

Inmultirea matricilor este distributiva A(B+C)=AB+AC, asociativa A(BC)=(AB)C, dar NU comutativa! AB ‚â† BA in general. Aceasta ordine conteaza enorm in deep learning.

‚ú®
Proprietati vizualizate ‚ñ∂
A(B + C) = AB + AC

‚úì Distributiva - functioneaza!

A(BC) = (AB)C

‚úì Asociativa - ordinea gruparii nu conteaza

AB ‚â† BA

‚ö†Ô∏è NU comutativa - ordinea factorilor conteaza!

(AB)·µÄ = B·µÄA·µÄ

üìù Transpusa produsului - ordinea se inverseaza!

üéÆ
Demo: Non-comutativitate ‚ñ∂
import numpy as np

A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])

print("AB =")
print(A @ B)
# [[19 22]
# [43 50]]

print("BA =")
print(B @ A)
# [[23 34]
# [31 46]]

print("AB == BA?", np.allclose(A@B, B@A)) # False!

‚ö†Ô∏è In deep learning, ordinea layerelor conteaza: Layer1(Layer2(x)) ‚â† Layer2(Layer1(x))

Comutativitatea Produsului Scalar

Desi inmultirea matricilor NU e comutativa, produsul scalar dintre doi vectori ESTE comutativ: x·µÄy = y·µÄx. Rezultatul e un scalar, iar scalarii sunt egali cu transpusa lor.

üìö
Demonstratie eleganta ‚ñ∂
üßÆ De ce x·µÄy = y·µÄx?
x·µÄy este un scalar (1√ó1)

Un scalar este egal cu transpusa sa: s = s·µÄ

x·µÄy = (x·µÄy)·µÄ = y·µÄ(x·µÄ)·µÄ = y·µÄx

Sistemul de Ecuatii Liniare: Ax = b

Ecuatia Ax = b este forma compacta a unui sistem de ecuatii liniare. A este matricea coeficientilor, x este vectorul necunoscutelor, b este vectorul rezultatelor. Rezolvarea acestei ecuatii sta la baza multor algoritmi ML!
