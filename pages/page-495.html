Cold-Start Problem

O limitare fundamentala a sistemelor de collaborative filtering este: cand un item nou sau un user nou este introdus, nu exista istorie de rating-uri pentru a evalua similaritatea cu alti users sau items. Aceasta se numeste problema cold-start.

12.5.1.1 Exploration Versus Exploitation

Cand facem recomandari utilizatorilor, apare o problema care depaseste supervised learning standard si intra in domeniul reinforcement learning. Problema este descrisa teoretic ca contextual bandits: vedem doar raspunsul utilizatorilor la itemele recomandate, nu si la celelalte iteme!
