Antrenam autoencoder pe patches 8x8 class PatchAutoencoder(nn.Module): def __init__(self, patch_size=8, hidden_dim=64): super().__init__() input_dim = 3 * patch_size * patch_size # 192 pentru RGB 8x8 self.encoder = nn.Sequential( nn.Linear(input_dim, hidden_dim), nn.ReLU() ) self.decoder = nn.Linear(hidden_dim, input_dim) def forward(self, x): # x: [batch, 3, 8, 8] x = x.flatten(1) # [batch, 192] z = self.encoder(x) # [batch, 64] return self.decoder(z).view(-1, 3, 8, 8) # 2. Dupa antrenare, folosim encoder-ul ca "kernel" # Practic: Conv2d(3, 64, 8) cu ponderile din encoder # Avantaj: antrenarea pe patches e MULT mai rapida # decat backprop prin intreaga retea!

Hubel si Wiesel: Descoperirea Celulelor Simple

CNN-urile sunt poate cel mai mare succes al AI-ului inspirat biologic. Istoria incepe cu neurostiinta: David Hubel si Torsten Wiesel (premiul Nobel 1981) au studiat cum neuronii din cortexul vizual al pisicilor raspund la stimuli vizuali. Descoperirea lor revolutionara: neuronii din zona V1 raspund selectiv la bare orientate in locatii specifice!

Celule Simple si Complexe

Hubel si Wiesel au identificat doua tipuri de neuroni: celule simple - raspund la bare la orientari SI pozitii specifice (ca neuronii dupa convolutie), si celule complexe - raspund la orientari specifice dar sunt mai tolerante la pozitie (ca dupa pooling). Aceasta ierarhie este fundamentul CNN-urilor!
