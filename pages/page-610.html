17.3 Markov Chain Monte Carlo Methods

Cand nu putem sample direct din p(x) sau folosi importance sampling eficient, apelam la MCMC. Construim un Markov chain a carui distributie stationara este p(x). Dupa suficienti pasi, starea chain-ului este aproximativ un sample din p.

# MCMC conceptual def mcmc_sample(target_p, transition, n_samples, burn_in): """ target_p: distributia tinta p(x) transition: T(x'|x) - propunere de tranzitie """ x = random_init() # Start aleator samples = [] for t in range(burn_in + n_samples): # Propune tranzitie x_new = transition(x) # Accept/reject (Metropolis-Hastings) if accept(x, x_new, target_p): x = x_new if t >= burn_in: samples.append(x) return samples
