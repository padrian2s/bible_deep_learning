12.2.1.2 Dataset Augmentation pentru Computer Vision

Object recognition este un task de clasificare care este deosebit de potrivit pentru dataset augmentation deoarece clasa este invarianta la atatea transformari geometrice ale input-ului. Pe langa translatii, rotatii si flip-uri, se folosesc si perturbari de culoare si distorsiuni geometrice nonliniare.

# Augmentari avansate pentru computer vision from torchvision import transforms advanced_augmentation = transforms.Compose([ # Geometric transforms transforms.RandomRotation(15), transforms.RandomResizedCrop(224, scale=(0.8, 1.0)), transforms.RandomHorizontalFlip(), # Color perturbation (Krizhevsky et al., 2012) transforms.ColorJitter( brightness=0.2, contrast=0.2, saturation=0.2, hue=0.1 ), # Nonlinear geometric distortions (LeCun et al., 1998b) transforms.RandomPerspective(distortion_scale=0.2), transforms.ToTensor(), transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225]) ])

Istoric: GMM-HMM Systems

Din anii 1980 pana in 2009-2012, sistemele ASR de stat-of-the-art combinau Hidden Markov Models (HMM) cu Gaussian Mixture Models (GMM). HMM-urile modelau secventa de foneme, iar GMM-urile asociau caracteristicile acustice cu foneme sau stari sub-fonemice.
