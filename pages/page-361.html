FARA layere FC!

Fully Convolutional Networks (FCN)

Arhitectura din dreapta (Fully Convolutional) este deosebit de interesanta: nu are NICIUN layer fully connected! Ultimul layer convolutional produce o feature map cu 1000 canale (cate unul per clasa). Global Average Pooling reduce fiecare canal la o singura valoare. Aceasta arhitectura: (1) are mult mai putini parametri, (2) poate fi aplicata pe imagini de orice dimensiune, (3) produce harti de probabilitate per-pixel (utile pentru segmentare).

import torch.nn as nn class FullyConvolutionalNet(nn.Module): """Arhitectura #3 - fara FC layers""" def __init__(self, num_classes=1000): super().__init__() self.features = nn.Sequential( nn.Conv2d(3, 64, 3, padding=1), nn.ReLU(), nn.MaxPool2d(4, 4), nn.Conv2d(64, 64, 3, padding=1), nn.ReLU(), nn.MaxPool2d(4, 4), ) # "FC layer" implementat ca Conv 1x1 self.classifier = nn.Conv2d(64, num_classes, 1) # Global Average Pooling self.gap = nn.AdaptiveAvgPool2d(1) def forward(self, x): x = self.features(x) # [B, 64, H/16, W/16] x = self.classifier(x) # [B, 1000, H/16, W/16] x = self.gap(x) # [B, 1000, 1, 1] return x.squeeze(-1).squeeze(-1) # [B, 1000] # Avantaj: functioneaza pe orice dimensiune! model = FullyConvolutionalNet() out_256 = model(torch.randn(1, 3, 256, 256)) # [1, 1000] out_512 = model(torch.randn(1, 3, 512, 512)) # [1, 1000]
