Aplicatii Pseudolikelihood

Pseudolikelihood este special utila pentru MRF-uri unde conditionalele depind doar de Markov blanket. Aceasta face calculul tractabil chiar si pentru modele cu mii de variabile. Besag (1975) a introdus metoda initial pentru modele spatiale.

# MRF pe grid 2D (ex: Ising model) # P(x_ij | rest) depinde doar de 4 vecini def pseudolikelihood_grid(x, W, b): """ PL pentru MRF pe grid """ log_pl = 0 for i in range(H): for j in range(W): neighbors = get_neighbors(i, j) # P(x_ij | neighbors) energy_0 = compute_local_energy(0, neighbors, W, b) energy_1 = compute_local_energy(1, neighbors, W, b) log_pl += x[i,j] * energy_1 - logsumexp([energy_0, energy_1]) return log_pl
