12.2.1 Preprocessing

Multe arii de aplicare necesita preprocessing sofisticat, dar computer vision necesita relativ putin. Imaginile trebuie standardizate astfel incat pixelii sa fie in acelasi range (ex: [0,1] sau [-1,1]). Amestecarea imaginilor cu range-uri diferite va esua antrenamentul!

üéÆ
Simulare: Preprocessing Standard ‚ñ∂
# Preprocessing standard pentru imagini import numpy as np def preprocess_image(img): # 1. Rescaling la range standard if img.max() > 1: img = img / 255.0 # [0, 255] -> [0, 1] # Sau pentru range [-1, 1]: # img = (img / 255.0) * 2 - 1 # 2. Resize la dimensiune standard (daca arhitectura cere) img = resize(img, (224, 224)) # 3. Mean subtraction (optional, AlexNet style) img = img - mean_across_training_set return img
AlexNet: Sistemul care a clasificat ImageNet a avut un singur pas de preprocessing - scaderea mediei pe pixeli din setul de antrenament (Krizhevsky et al., 2012).

Dataset Augmentation

Dataset augmentation poate fi vazuta ca o forma de preprocessing doar pe setul de training. Este o metoda excelenta pentru reducerea erorii de generalizare. La test time, o idee similara este sa aratam modelului mai multe versiuni ale aceluiasi input si sa facem votare - abordare de tip ensemble.
