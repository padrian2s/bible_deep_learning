De ce Gaussiana e atat de populara?

Distributia normala este alegerea implicita in lipsa cunostintelor prealabile, din doua motive fundamentale. In primul rand, multe distributii reale sunt aproape normale - Teorema Limitei Centrale ne arata ca suma multor variabile independente converge catre normala. import numpy as np # Demonstratie CLT: suma variabilelor converge la normala # Chiar daca variabilele originale NU sunt normale! np.random.seed(42) n_experiments = 10000 # Suma de zaruri (distributie uniforma discreta) def clt_demonstration(n_dice): """Arunca n zaruri, calculeaza suma""" sums = [] for _ in range(n_experiments): rolls = np.random.randint(1, 7, n_dice) sums.append(rolls.sum()) return np.array(sums) # Cu cat mai multe zaruri, cu atat mai "normala" e distributia for n in [1, 2, 5, 30]: sums = clt_demonstration(n) # Standardizam pentru comparatie standardized = (sums - sums.mean()) / sums.std() # Testam normalitatea (p > 0.05 = probabil normala) from scipy.stats import normaltest _, p_value = normaltest(standardized) print(f"{n} zaruri: mean={sums.mean():.1f}, std={sums.std():.2f}, " f"normal? p={p_value:.4f}") # Output: cu n=30, p-value devine mare (distribuita normal)
