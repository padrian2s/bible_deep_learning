5.1.3 Experienta, E

Algoritmii ML pot fi categorizati in unsupervised sau supervised in functie de ce tip de experienta au in timpul invatarii. Majoritatea algoritmilor din aceasta carte experimenteaza un intreg dataset - o colectie de exemple, numite si data points. Unul dintre cele mai vechi dataset-uri studiate este Iris dataset (Fisher, 1936) - 150 de masuratori ale plantelor de iris cu 4 features fiecare.

Unsupervised Learning

Algoritmii unsupervised learning experimenteaza un dataset cu multe features si invata proprietati utile ale structurii acestuia. In deep learning, vrem de obicei sa invatam intreaga distributie de probabilitate care a generat dataset-ul (explicit pentru estimare de densitate, implicit pentru sinteza sau denoising). Alte task-uri unsupervised includ clustering - impartirea dataset-ului in clustere de exemple similare.

Supervised Learning

Algoritmii supervised learning experimenteaza un dataset cu features, dar fiecare exemplu are si un label sau target asociat. Iris dataset poate fi studiat supervised - invatam sa clasificam plantele in trei specii bazat pe masuratori. Informal: unsupervised observa exemple de x si invata p(x), pe cand supervised observa perechi (x, y) si invata p(y | x). Termenul "supervised" vine de la ideea ca un instructor furnizeaza target-ul y.
