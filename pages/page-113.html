Capitolul 5 - Machine Learning Basics

Ce este Machine Learning?

Deep learning este un tip specific de machine learning. Pentru a intelege deep learning bine, trebuie sa avem o intelegere solida a principiilor de baza ale machine learning. Acest capitol ofera un curs scurt despre cele mai importante principii generale care vor fi aplicate pe parcursul cartii. Cititorii incepatori ar trebui sa consulte manuale mai comprehensive precum Murphy (2012) sau Bishop (2006).
