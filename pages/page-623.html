18.2 Stochastic Maximum Likelihood and Contrastive Divergence

Contrastive Divergence (CD) este o aproximare populara pentru antrenarea EBM-urilor. In loc sa rulam MCMC pana la convergenta, pornim chain-ul de la datele de training si rulam doar k pasi (de obicei k=1). Aceasta ofera un gradient biased dar functional.

def CD_k(data, model, k=1): """ Contrastive Divergence cu k pasi Gibbs """ # Positive phase: start from data v_pos = data h_pos = sample_h_given_v(v_pos, model) # Negative phase: k pasi de Gibbs v_neg = v_pos for _ in range(k): h_neg = sample_h_given_v(v_neg, model) v_neg = sample_v_given_h(h_neg, model) # Gradient aproximat grad_W = outer(v_pos, h_pos) - outer(v_neg, h_neg) return grad_W
