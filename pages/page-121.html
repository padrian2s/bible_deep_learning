Conexiunea Unsupervised-Supervised

Putem rezolva problema supervised p(y | x) folosind tehnici unsupervised: invatam distributia comuna p(x, y) si apoi inferim:

p(y | x) = p(x, y) / Œ£·µß p(x, y')

Desi nu sunt concepte complet distincte, unsupervised si supervised ajuta la categorizarea algoritmilor. Traditional, regresie si clasificare = supervised, estimare de densitate = unsupervised.

Design Matrix

Un mod comun de a descrie un dataset este cu o design matrix. O design matrix este o matrice care contine un exemplu diferit in fiecare rand. Fiecare coloana corespunde unei features diferite.
