12.4 Natural Language Processing

NLP (Natural Language Processing) este folosirea limbajelor umane, cum ar fi engleza sau romana, de catre computere. Programele citesc si emit limbaj natural. Aplicatiile includ machine translation, question answering, sentiment analysis si multe altele. Multe aplicatii NLP se bazeaza pe language models.

De ce e dificil NLP: Limbajele naturale sunt adesea ambigue si sfideaza descrierea formala. Tehnicile generice de retele neurale pot fi aplicate cu succes, dar sunt necesare strategii domain-specific pentru scalare la aplicatii mari.

12.4.1 n-grams

Un language model defineste o distributie de probabilitate peste secvente de tokens intr-un limbaj natural. Token-ul poate fi un cuvant, caracter sau chiar byte. Primele modele de succes se bazau pe n-grams - secvente de n tokens consecutive.
