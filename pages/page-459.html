12.1.1 Implementari CPU Rapide

Traditional, retelele neurale erau antrenate folosind CPU-ul unei singure masini. Astazi, aceasta abordare este considerata insuficienta - folosim GPU computing sau CPU-uri in retea. Totusi, optimizarea atenta pentru CPU-uri specifice poate aduce imbunatatiri semnificative.

12.1.2 Implementari GPU

Majoritatea implementarilor moderne de retele neurale se bazeaza pe GPU-uri (Graphics Processing Units). Aceste componente hardware specializate au fost dezvoltate initial pentru aplicatii grafice, dar caracteristicile lor de performanta sunt benefice si pentru retelele neurale.

âœ¨
Vizualizare: CPU vs GPU â–¶
CPU
  • Clock speed mare
  • Branching sofisticat
  • Cache mare
  • Putine core-uri (~8-16)
GPU
  • Paralelism masiv
  • Bandwidth memorie mare
  • Clock speed mai mic
  • Mii de core-uri (~1000+)

Retelele neurale beneficiaza de paralelismul GPU deoarece neuronii dintr-un layer pot fi procesati independent!
