Introducere

De ce avem nevoie de Probabilitati in Machine Learning?

Teoria probabilitatilor este un cadru matematic pentru reprezentarea afirmatiilor incerte. Ne ofera o modalitate de a cuantifica incertitudinea si axiome pentru a deriva noi afirmatii incerte. In inteligenta artificiala, folosim probabilitatile in doua moduri majore: pentru a proiecta algoritmi care calculeaza expresii derivate din distributii de probabilitate, si pentru a analiza teoretic comportamentul sistemelor AI propuse.

import numpy as np # 1. Design: Clasificare probabilistica def classify_probabilistic(x, model): # Output: probabilitati pentru fiecare clasa probs = model.predict_proba(x) return probs # [0.87, 0.13] - nu doar "spam" # 2. Analiza: Estimare eroare prin bootstrapping def analyze_model_uncertainty(model, X, y, n_bootstrap=1000): scores = [] for _ in range(n_bootstrap): idx = np.random.choice(len(X), len(X), replace=True) score = model.score(X[idx], y[idx]) scores.append(score) return { 'mean': np.mean(scores), 'std': np.std(scores), # Incertitudinea estimarii 'ci_95': np.percentile(scores, [2.5, 97.5]) }
