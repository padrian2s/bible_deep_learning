General Purpose GPU (GP-GPU)

Popularitatea placilor grafice pentru antrenarea retelelor neurale a explodat dupa aparitia GP-GPU-urilor. Acestea pot executa cod arbitrar, nu doar subrutine de rendering. Limbajul CUDA al NVIDIA a oferit o modalitate convenabila de a scrie acest cod in C.

WARP - grup de thread-uri
Warps: Thread-urile GPU sunt impartite in grupuri mici numite warps. Fiecare thread dintr-un warp executa aceeasi instructiune in fiecare ciclu. Daca branch-urile difera, caile de cod diferite sunt traversate secvential, nu in paralel.

// Exemplu simplificat de kernel CUDA pentru forward pass __global__ void neural_layer_forward( float* input, float* weights, float* output, int input_size, int output_size) { // Fiecare thread proceseaza un neuron de output int neuron_idx = blockIdx.x * blockDim.x + threadIdx.x; if (neuron_idx < output_size) { float sum = 0.0f; for (int i = 0; i < input_size; i++) { sum += input[i] * weights[neuron_idx * input_size + i]; } output[neuron_idx] = tanh(sum); // Activare } }

Mii de neuroni sunt procesati simultan, fiecare de un thread separat!
