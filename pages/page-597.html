Sampling din Modele Nedirectionate

Pentru MRF-uri, sampling-ul direct nu este posibil din cauza partition function-ului intractabil. Folosim metode MCMC (Markov Chain Monte Carlo), in special Gibbs sampling: iteram prin variabile si re-sample-uim fiecare din distributia conditionala date toate celelalte.

def gibbs_sampling(mrf, num_samples, burn_in=1000): """ Gibbs sampling pentru MRF """ # Initializare aleatoare state = random_init(mrf.variables) samples = [] for t in range(burn_in + num_samples): # Iteram prin variabile for var in mrf.variables: # P(var | toate celelalte) neighbors = mrf.markov_blanket(var) state[var] = sample_conditional( var, state, neighbors ) if t >= burn_in: samples.append(state.copy()) return samples
