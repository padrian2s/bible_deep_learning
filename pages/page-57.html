2.7 Eigendecomposition - Descompunerea Matricelor

Asa cum numerele intregi pot fi descompuse in factori primi (12 = 2√ó2√ó3), matricele pot fi descompuse in parti fundamentale. Eigendecomposition ne arata "directiile naturale" ale unei transformari matriciale!

Eigenvector si Eigenvalue

Un eigenvector v al matricei A este un vector special: cand il inmultim cu A, el NU isi schimba directia, doar se scaleaza! Factorul de scalare este eigenvalue Œª: Av = Œªv.

Vizualizare: Av = Œªv ‚ñ∂
Av = Œªv
Œª > 1

Vector se LUNGESTE in directia v

0 < Œª < 1

Vector se SCURTEAZA in directia v

Œª < 0

Vector se INVERSEAZA si scaleaza

Cod: Calculul eigenvalues ‚ñ∂
import numpy as np

A = np.array([[3, 1],
              [1, 3]])

# Calculeaza eigenvalues si eigenvectors
eigenvalues, eigenvectors = np.linalg.eig(A)

print("Eigenvalues:", eigenvalues) # [4. 2.]
print("Eigenvectors:")
print(eigenvectors)
# [[ 0.707 -0.707]
# [ 0.707 0.707]]

# Verifica: A @ v = Œª @ v
v = eigenvectors[:, 0]
print(A @ v) # [2.83, 2.83]
print(eigenvalues[0] * v) # [2.83, 2.83] ‚úì

Formula Eigendecomposition

Daca A are n eigenvectors liniar independenti, o putem scrie ca: A = V diag(Œª) V‚Åª¬π. V contine eigenvectorii ca coloane, diag(Œª) are eigenvalues pe diagonala.
