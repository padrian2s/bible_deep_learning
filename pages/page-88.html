Self-Information si Entropia Shannon

Ce este Teoria Informatiei?

Teoria informatiei este o ramura a matematicii aplicata care cuantifica cat de multa informatie este prezenta intr-un semnal. A fost inventata pentru a studia transmiterea mesajelor peste canale zgomotoase (ex: comunicatii radio). In ML, folosim teoria informatiei pentru a caracteriza distributii de probabilitate si a masura similaritatea dintre ele. Referinte fundamentale: Cover & Thomas (2006), MacKay (2003).

Intuitia din spatele informatiei

Intuitia de baza: aflarea ca un eveniment improbabil s-a intamplat este mai informativa decat aflarea ca un eveniment probabil s-a intamplat. Mesajul "soarele a rasarit azi dimineata" nu ne spune nimic nou. Dar "a fost o eclipsa de soare azi dimineata" este foarte informativ! Vrem sa cuantificam aceasta idee matematic.

Informatie MARE

Proprietatile Dorite ale Informatiei

Vrem ca masura informatiei sa satisfaca: (1) Evenimentele cu probabilitate 1 (garantate) au informatie 0. (2) Evenimentele mai putin probabile au informatie mai mare. (3) Informatia este aditiva pentru evenimente independente - aflarea a doua lucruri independente da de doua ori mai multa informatie decat aflarea unuia singur.

‚ú®
Vizualizare: Cele 3 proprietati ‚ñ∂
1. P(x) = 1 ‚Üí I(x) = 0

Evenimentele garantate nu aduc informatie noua

2. P(x) mic ‚Üí I(x) mare

Cu cat mai rar, cu atat mai informativ

3. I(x, y) = I(x) + I(y) pentru x ‚ä• y

Informatia e aditiva pentru evenimente independente

Self-Information (Auto-Informatie)

Singura functie care satisface toate cele 3 proprietati este self-information: I(x) = -log P(x). Folosim logaritmul natural (baza e), iar unitatea de masura se numeste nat. Daca folosim log baza 2, unitatea e bit. Un nat este cantitatea de informatie obtinuta prin observarea unui eveniment cu probabilitate 1/e ‚âà 0.368.

üéÆ
Cod: Self-Information ‚ñ∂
import numpy as np def self_information_nats(p): """I(x) = -log(p) in nats""" return -np.log(p) def self_information_bits(p): """I(x) = -log2(p) in bits""" return -np.log2(p) # Exemple print("Self-Information:") print(f"{'P':^10} {'I (nats)':^12} {'I (bits)':^12}") print("-" * 36) for p in [1.0, 0.5, 0.25, 0.1, 0.01, 0.001]: i_nat = self_information_nats(p) i_bit = self_information_bits(p) print(f"{p:^10.3f} {i_nat:^12.3f} {i_bit:^12.3f}") # Verificare aditivitate pentru evenimente independente # P(A si B) = P(A) * P(B) pentru independente p_a, p_b = 0.5, 0.25 p_ab = p_a * p_b print(f"\nAditivitate: I(A) + I(B) = {self_information_nats(p_a) + self_information_nats(p_b):.3f}") print(f" I(A,B) = {self_information_nats(p_ab):.3f}")

Nats vs Bits ‚ñ∂
Unitati de masura:

Bit (log baza 2) - informatia obtinuta prin aflarea rezultatului unei aruncari de moneda corecta. Folosit in compresie, comunicatii.

Nat (log natural) - folosit in ML pentru ca derivatele sunt mai simple. 1 nat ‚âà 1.443 bits.

Conversia: 1 nat = 1/ln(2) bits ‚âà 1.443 bits
