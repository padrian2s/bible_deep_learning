5.11.3 Manifold Learning

Un concept important care sta la baza multor idei in ML este cel de manifold. Un manifold este o regiune conectata. Matematic, este un set de puncte, asociat cu o vecinatate in jurul fiecarui punct. De la orice punct dat, manifold-ul local pare a fi un spatiu Euclidian. In viata de zi cu zi, experimentam suprafata lumii ca un plan 2D, dar de fapt este un manifold sferic in spatiul 3D.

Manifolds in Machine Learning

Desi exista o definitie matematica formala, in ML termenul "manifold" este folosit mai lejer pentru a desemna un set conectat de puncte care poate fi aproximat bine considerand doar un numar mic de degrees of freedom sau dimensiuni, embedded in spatiu de dimensiune mai mare. Fiecare dimensiune corespunde unei directii locale de variatie. Vezi Figura 5.11 pentru un exemplu de date de training aproape de un manifold 1D embedded in spatiu 2D.

Figura 5.11: Manifold 1D in 2D

Date sample-uite dintr-o distributie in spatiu 2D care este de fapt concentrata aproape de un manifold 1D, ca un string rasucit. Linia solida indica manifold-ul pe care learner-ul ar trebui sa-l infereze. In contextul ML, permitem dimensionalitatea manifold-ului sa varieze de la punct la punct - de exemplu, o cifra 8 are o singura dimensiune in majoritatea locurilor, dar doua dimensiuni la intersectia din centru.

import numpy as np np.random.seed(42) # Generam date pe un manifold 1D (spirala) in 2D t = np.linspace(0, 4*np.pi, 100) x1_manifold = t * np.cos(t) x2_manifold = t * np.sin(t) # Adaugam noise mic perpendicular pe manifold noise = 0.3 x1 = x1_manifold + noise * np.random.randn(100) x2 = x2_manifold + noise * np.random.randn(100) print("Datele par 2D, dar:") print(f" Dimensiune aparenta: 2") print(f" Dimensiune intrinseca: 1 (de-a lungul spiralei)") print() print("Statistici:") print(f" Varianta totala 2D: {np.var(x1) + np.var(x2):.2f}") print() print("=> Manifold learning gaseste") print(" structura low-dimensional!") # PCA ar gasi 2 componente importante # Dar manifold e curbat - avem nevoie de # metode nonlineare (t-SNE, UMAP, autoencoders)
