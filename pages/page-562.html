Half-Spaces si Regiuni

Capitolul 15 - Figura 15.7

Figura 15.7: Reprezentare Distribuita cu Features Binare

Ilustratie a modului in care un algoritm de invatare distribuit imparte spatiul de input. Avem 3 features binare h₁, h₂, h₃, fiecare definit prin thresholding-ul unei transformari liniare. Fiecare feature imparte R² in doua half-spaces. Reprezentarea ca intreg ia o valoare unica la fiecare intersectie a acestor half-spaces. Exemplu: h = [1,1,1]ᵀ corespunde regiunii h₁⁺ ∩ h₂⁺ ∩ h₃⁺.

n hyperplane-uri → O(n^d) regiuni in d dimensiuni

Avantajul Exponential

Reprezentarea distribuita imparte spatiul prin intersectia half-spaces, nu prin atribuirea de regiuni disjuncte. Cu n features in d dimensiuni, putem distinge O(n^d) regiuni! Nearest neighbors cu n exemple distinge doar n regiuni. Reprezentarea distribuita poate astfel distinge exponential mai multe regiuni - baza puterii retelelor neurale adanci.
