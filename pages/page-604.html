Capitolul 16 - Final

Sumar Capitolul 16

Modelele probabilistice structurate ofera un limbaj puternic pentru descrierea distributiilor complexe. Deep learning extinde aceste idei cu variabile latente masive, reprezentari distribuite, si retele neurale adanci pentru conditionale. Intelegerea modelelor grafice este fundamentala pentru deep generative models.
