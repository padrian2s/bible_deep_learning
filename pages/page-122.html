Labels in Supervised Learning

In cazul supervised learning, exemplul contine si un label sau target pe langa features. De exemplu, pentru recunoasterea obiectelor din fotografii, specificam obiectul din fiecare fotografie cu un cod numeric (0=persoana, 1=masina, 2=pisica, etc.). Cand lucram cu design matrix X, furnizam si un vector de labels y, unde yi este label-ul pentru exemplul i. Uneori label-ul poate fi mai mult decat un singur numar - de exemplu, pentru speech recognition, label-ul este o secventa de cuvinte.

5.1.4 Exemplu: Regresia Liniara

Definitia noastra de algoritm ML ca unul care imbunatateste performanta la un task prin experienta este oarecum abstracta. Pentru a face lucrurile concrete, prezentam un exemplu simplu: regresia liniara. Vom reveni la acest exemplu pe parcursul cartii pentru a introduce concepte ML. Terminologie: Parametri ‚ñ∂
Parametri (Weights)

Parametrii sunt valori care controleaza comportamentul sistemului. In acest caz, wi este coeficientul cu care inmultim feature xi inainte de a suma contributiile tuturor features.

  • wi pozitiv: feature creste predictia
  • wi negativ: feature scade predictia
  • |wi| mare: feature are impact puternic
