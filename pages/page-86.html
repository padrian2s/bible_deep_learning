Bayes in Machine Learning

Regula lui Bayes e fundamentul multor tehnici ML. Formula se deriva direct din definitia probabilitatii conditionate, dar e util sa o cunoastem pe nume. In ML, adesea x sunt parametrii modelului si y sunt datele. Bayesian inference inseamna calculul posterior-ului P(parametri|date) din prior P(parametri) si likelihood P(date|parametri).

Masura Zero si "Aproape Peste Tot"

Un set are masura zero daca nu ocupa "volum" in spatiul pe care il masuram. De exemplu, in ‚Ñù¬≤, o linie are masura zero (dar un poligon plin are masura pozitiva). Un punct individual are mereu masura zero. Orice reuniune numarabila de seturi de masura zero are tot masura zero (ex: multimea numerelor rationale are masura zero in ‚Ñù!).

"Aproape Peste Tot" (Almost Everywhere)

O proprietate este adevarata aproape peste tot daca exceptiile formeaza un set de masura zero. De exemplu, "PDF-urile sunt continue aproape peste tot" inseamna ca pot avea discontinuitati, dar doar pe un set de masura zero. In practica, rezultatele care sunt adevarate "aproape peste tot" sunt la fel de bune ca cele adevarate "peste tot" - exceptiile sunt neglijabile probabilistic.

import numpy as np # Functie cu discontinuitate la x=0 def f_discontinuous(x): """f(x) = 0 pentru x < 0, 1 pentru x >= 0""" return (x >= 0).astype(float) # Esantionare uniforma din [-1, 1] np.random.seed(42) samples = np.random.uniform(-1, 1, 1000000) # Cat de des "nimereste" exact x = 0? exact_zero = np.sum(samples == 0.0) print(f"Esantioane exact la x=0: {exact_zero}") # = 0! # Probabilitatea de a nimeri un punct specific e 0 # Discontinuitatile pe seturi de masura zero nu conteaza # pentru integrale (sperante, probabilitati pe intervale) # Integrala e aceeasi ca pentru step function continua: mean_f = f_discontinuous(samples).mean() expected = 0.5 # P(x >= 0) pentru Uniform(-1, 1) print(f"\nE[f(x)] empiric: {mean_f:.4f}") print(f"E[f(x)] teoretic: {expected}")
