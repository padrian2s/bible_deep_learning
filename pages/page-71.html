3.2 Variabile Aleatoare

O variabila aleatoare este o variabila care poate lua diferite valori aleatoriu. Notam variabila cu litera mica (ex: x) si valorile pe care le poate lua cu litera mica cursiva (ex: x‚ÇÅ, x‚ÇÇ). Pentru vectori, folosim x pentru variabila si x pentru valori. O variabila aleatoare singura e doar o descriere a starilor posibile - trebuie cuplata cu o distributie de probabilitate care specifica cat de probabila e fiecare stare.

‚ú®
Vizualizare: Notatie ‚ñ∂
Scalari
Variabila: x
Valori: x‚ÇÅ, x‚ÇÇ, ...

Ex: x = varsta, valori posibile: 0, 1, 2, ...

Vectori
Variabila: x
Valori: x

Ex: x = [varsta, greutate, inaltime]

üéÆ
Cod: Variabile aleatoare in Python ‚ñ∂
import numpy as np from scipy import stats # Variabila aleatoare scalara # x = rezultatul unui zar (valori posibile: 1-6) dice = stats.randint(1, 7) # Definim distributia samples = dice.rvs(size=10) # Esantionam valori print(f"Esantioane zar: {samples}") # Variabila aleatoare vectoriala # x = [temperatura, umiditate] mean = [20, 60] # Grade Celsius, procent cov = [[5, 2], [2, 100]] # Covariance weather = stats.multivariate_normal(mean, cov) sample = weather.rvs() print(f"Esantion vreme: temp={sample[0]:.1f}¬∞C, umid={sample[1]:.0f}%")

Variabile Discrete vs Continue

Variabilele aleatoare pot fi discrete sau continue. O variabila discreta are un numar finit sau numarabil infinit de stari - acestea nu trebuie sa fie neaparat numere intregi, pot fi si stari nominale (ex: "rosu", "verde", "albastru"). O variabila continua este asociata cu valori reale.

3.3 Distributii de Probabilitate

O distributie de probabilitate descrie cat de probabila e fiecare stare (sau regiune de stari) a unei variabile aleatoare. Modul in care descriem distributia depinde de tipul variabilei: pentru variabile discrete folosim functia de masa de probabilitate (PMF), iar pentru variabile continue folosim functia de densitate de probabilitate (PDF).

‚ú®
Vizualizare: PMF vs PDF ‚ñ∂
PMF - Probability Mass Function

Pentru variabile DISCRETE

P(x = k)

"Probabilitatea ca x sa fie exact k"

Valori intre 0 si 1

PDF - Probability Density Function

Pentru variabile CONTINUE

p(x) sau f(x)

"Densitatea probabilitatii la punctul x"

Poate fi > 1!

3.3.1 Variabile Discrete si Functia de Masa (PMF)

O distributie de probabilitate peste variabile discrete poate fi descrisa folosind o functie de masa de probabilitate (PMF). Notam PMF cu P majuscul. PMF-ul mapeaza fiecare stare a variabilei la probabilitatea ca variabila sa ia acea stare. Probabilitatea ca x = x‚ÇÅ se scrie P(x = x‚ÇÅ) sau prescurtat P(x‚ÇÅ).

üéÆ
Cod: PMF pentru un zar ‚ñ∂
import numpy as np import matplotlib.pyplot as plt # PMF pentru un zar corect def pmf_dice(k): """P(X = k) pentru zar cu 6 fete""" if k in [1, 2, 3, 4, 5, 6]: return 1/6 return 0 # Verificam proprietatile PMF values = [1, 2, 3, 4, 5, 6] probs = [pmf_dice(k) for k in values] print("PMF pentru zar:") for k, p in zip(values, probs): print(f" P(X = {k}) = {p:.4f}") print(f"\nSuma probabilitatilor: {sum(probs):.4f}") # = 1.0 print(f"Toate >= 0: {all(p >= 0 for p in probs)}") # True
üìö
Notatie importanta ‚ñ∂
Conventii de notatie:
  • P(x) - PMF, notam distributia cu variabila (P(x) ‚â† P(y) de obicei)
  • P(x = x‚ÇÅ) - Probabilitatea exacta ca x sa ia valoarea x‚ÇÅ
  • x ~ P(x) - "x este distribuit conform P"
  • P(x, y) - Distributie comuna (joint) peste x SI y
