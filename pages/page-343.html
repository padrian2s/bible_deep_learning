Curriculum Learning: O Metoda de Continuare

Bengio et al. (2009) observa ca curriculum learning poate fi interpretat ca o metoda de continuation. In loc sa modificam functia cost, modificam distributia datelor: incepem cu exemple uoare, apoi adaugam exemple dificile.

Khan et al. (2011) au verificat ca curriculum learning e consistent cu modul in care oamenii predau. Profesorii arata exemple "prototipice" intai.

Aplicatii in NLP si Computer Vision

Curriculum learning a avut succes in multe domenii: NLP (Spitkovsky et al., 2010; Collobert et al., 2011; Mikolov et al., 2011), computer vision (Kumar et al., 2010; Lee & Grauman, 2011), si alte aplicatii.

Stochastic Curriculum

Zaremba & Sutskever (2014) au gasit ca un stochastic curriculum functioneaza mai bine decat unul deterministic: mereu un mix de exemple uoare si dificile, dar proportia de exemple dificile creste gradual.

Implementare â–¶
# Stochastic Curriculum Learning def get_curriculum_batch(epoch, max_epochs, dataset): # Proportia de exemple dificile creste cu epoch difficulty_threshold = epoch / max_epochs # Sortam dupa dificultate easy = [x for x in dataset if x.difficulty < 0.5] hard = [x for x in dataset if x.difficulty >= 0.5] # Mix stochastic n_hard = int(batch_size * difficulty_threshold) n_easy = batch_size - n_hard batch = random.sample(easy, n_easy) + \ random.sample(hard, n_hard) return shuffle(batch)

Un curriculum deterministic (doar exemple usoare la inceput) nu a aratat imbunatatiri fata de training normal.
