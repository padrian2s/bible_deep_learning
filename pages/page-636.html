NCE Invata log Z

O proprietate remarcabila a NCE: poate invata log Z ca parametru! Adaugam c = log Z ca parametru trainabil. La convergenta, c converge la log Z adevarat. Aceasta permite evaluarea log-likelihood-ului!

Z ca Parametru ▶

Model NCE:

log p(x) = log p̃(x;θ) - c

Unde c este un parametru trainabil.

La optim: c → log Z(θ)
