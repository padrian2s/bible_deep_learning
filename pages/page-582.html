16.2.2 Undirected Models (Markov Random Fields)

Modelele nedirectionate (Markov Random Fields sau MRF) folosesc grafuri cu muchii fara directie. Sunt potrivite pentru corelatii simetrice intre variabile. Distributia se exprima prin clique potentials - functii definite pe grupuri de noduri complet conectate (clique-uri). Spre deosebire de modelele directionate, MRF necesita o constanta de normalizare Z.

import torch import torch.nn as nn class MarkovRandomField(nn.Module): """ MRF simplu pentru 3 variabile binare. Graf: A --- B --- C (lant) Clique-uri: {A,B}, {B,C} """ def __init__(self): super().__init__() self.phi_ab = nn.Parameter(torch.randn(2, 2)) self.phi_bc = nn.Parameter(torch.randn(2, 2)) self.psi_a = nn.Parameter(torch.randn(2)) self.psi_b = nn.Parameter(torch.randn(2)) self.psi_c = nn.Parameter(torch.randn(2)) def unnormalized_prob(self, a, b, c): """Calculeaza p_tilde(a,b,c) = phi_ab(a,b) * phi_bc(b,c) * psi(a) * psi(b) * psi(c)""" pairwise = torch.exp(self.phi_ab[a, b] + self.phi_bc[b, c]) unary = torch.exp(self.psi_a[a] + self.psi_b[b] + self.psi_c[c]) return pairwise * unary def partition_function(self): """Calculeaza Z = sum over all configurations""" Z = 0.0 for a in [0, 1]: for b in [0, 1]: for c in [0, 1]: Z += self.unnormalized_prob(a, b, c) return Z def log_prob(self, a, b, c): """log P(a,b,c) = log p_tilde - log Z""" log_unnorm = torch.log(self.unnormalized_prob(a, b, c)) log_Z = torch.log(self.partition_function()) return log_unnorm - log_Z mrf = MarkovRandomField() print("Partition function Z:", mrf.partition_function().item()) print("P(0,0,0):", torch.exp(mrf.log_prob(0, 0, 0)).item()) print("P(1,1,1):", torch.exp(mrf.log_prob(1, 1, 1)).item())

Aplicatii: CRF, Ising Model, Image Segmentation â–¶
Ce este un Clique?

Un clique este un subset de noduri unde fiecare nod este conectat cu toate celelalte din subset. Clique-ul maximal nu poate fi extins cu niciun alt nod adiacent tuturor.

CRF (Conditional Random Fields)

Folosite in NLP pentru sequence labeling (NER, POS tagging). Modelul este un MRF conditionat de input x: P(y|x).

Ising Model

Model din fizica pentru ferromagnetism. Fiecare spin +1/-1 interactioneaza cu vecinii. Energie: E = -J * sum(s_i * s_j).

Image Segmentation

MRF pe grid de pixeli - fiecare pixel interactioneaza cu vecinii sai. Potentialele incurajeaza pixelii vecini sa aiba aceeasi eticheta.

Atentie: Problema lui Z

Spre deosebire de modelele directionate, MRF-urile necesita calculul constantei de normalizare Z = sum_x exp(-E(x)). Aceasta suma este de obicei intractabila (exponentiala in numarul de variabile), necesitand metode aproximative precum MCMC sau variational inference.
