Ce sunt Retelele Convolutionale (CNN)?

Retelele convolutionale (LeCun, 1989), cunoscute si ca CNN-uri, sunt un tip specializat de retea neurala pentru procesarea datelor cu topologie de tip grila. Exemple includ serii temporale (grila 1D de esantioane la intervale regulate) si imagini (grila 2D de pixeli). CNN-urile au avut un succes extraordinar in aplicatii practice, fiind fundamentale pentru recunoasterea de imagini, detectia obiectelor si multe alte sarcini de viziune computationala.

Operatia de Convolutie

Numele "retea neurala convolutionala" indica faptul ca reteaua foloseste o operatie matematica numita convolutie. Convolutia este un tip specializat de operatie liniara. Retelele convolutionale sunt pur si simplu retele neurale care folosesc convolutia in loc de inmultirea generala de matrice in cel putin unul din straturile lor.

üéÆ
Simulare: Convolutie 1D simpla ‚ñ∂
import numpy as np # Semnal de intrare (ex: serie temporala) signal = np.array([1, 2, 3, 4, 5, 6, 7, 8]) # Kernel (filtru) - detecteaza schimbari kernel = np.array([1, 0, -1]) # Convolutie 1D output = np.convolve(signal, kernel, mode='valid') print("Input:", signal) print("Kernel:", kernel) print("Output:", output) # Output: [2, 2, 2, 2, 2, 2] - diferente constante!
Ideea cheie: Kernelul [1, 0, -1] calculeaza diferenta intre elementul curent si cel de acum 2 pozitii - detecteaza "derivata" semnalului!
