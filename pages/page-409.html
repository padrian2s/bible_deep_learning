10.3 Bidirectional RNNs

Retele Recurente Bidirectionale

Figura 10.11 - Bidirectional RNN

RNN-urile discutate pana acum sunt cauzale: starea la timpul t depinde doar de trecut. Dar pentru multe aplicatii (speech recognition, tagging), predictia la t poate depinde si de viitor! Bidirectional RNN rezolva aceasta problema.

De ce Bidirectional?

In speech recognition, interpretarea unui fonem poate depinde de fonemele urmatoare (coarticulare). In NLP, intelesul unui cuvant depinde adesea de contextul din ambele parti. BiRNN-ul permite output-ului sa vada TOATA secventa.

Cod: BiRNN in PyTorch ‚ñ∂ 
import torch.nn as nn

# BiRNN in PyTorch - super simplu!
birnn = nn.RNN(
    input_size=100,
    hidden_size=256,
    bidirectional=True  # Magic flag!
)

# Output size = 2 * hidden_size
# Concateneaza forward si backward states
x = torch.randn(seq_len, batch, 100)
output, hidden = birnn(x)
# output shape: (seq_len, batch, 512)  # 256*2
