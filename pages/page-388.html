Retele Neuronale Recurente si Recursive

Ce sunt Retelele Neuronale Recurente (RNN)?

RNN-urile sunt o familie de retele neuronale specializate pentru procesarea datelor secventiale. Asa cum retelele convolutionale (CNN) sunt optimizate pentru grile de valori (imagini), RNN-urile sunt optimizate pentru secvente de valori x(1), ..., x(œÑ). Pot procesa secvente de lungime variabila, ceea ce le face perfecte pentru text, audio, sau serii temporale.

Parameter Sharing - Cheia RNN-urilor

Ideea revolutionara din anii 1980: in loc sa ai parametri separati pentru fiecare pozitie din secventa, imparti aceiasi parametri pe toate pozitiile. Astfel, modelul poate generaliza la secvente de orice lungime si poate recunoaste pattern-uri indiferent unde apar in secventa.

‚ú®
Vizualizare: De ce conteaza ‚ñ∂

Exemplu: "Am fost in Nepal in 2009" vs "In 2009, am fost in Nepal"

Fara parameter sharing:

Modelul ar trebui sa invete separat ca "2009" este un an la pozitia 6 SI la pozitia 2. Imposibil de scalat!

Cu parameter sharing:

Acelasi "detector de an" functioneaza la ORICE pozitie. Invata o data, aplica oriunde!

Cod: Simplu RNN in PyTorch ‚ñ∂ 
import torch.nn as nn

# RNN simplu - aceiasi W pentru toate timesteps
rnn = nn.RNN(
    input_size=10,    # dimensiune input
    hidden_size=20,   # dimensiune stare ascunsa
    num_layers=1      # un singur layer
)

# Secventa de lungime 5, batch 3
x = torch.randn(5, 3, 10)  # (seq_len, batch, input)
h0 = torch.zeros(1, 3, 20) # stare initiala

output, hn = rnn(x, h0)
# output: (5, 3, 20) - output la fiecare pas
# hn: (1, 3, 20) - starea finala
